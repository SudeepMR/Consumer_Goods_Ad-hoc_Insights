## Project Overview

This project focuses on analyzing consumer goods sales data to generate actionable business insights related to product performance, regional trends, customer behavior, and operational KPIs.

The objective was to support ad-hoc business decision-making through data analysis, visualization, and reporting using SQL, Power BI, and Excel.

The analysis involved:
- Data extraction and transformation
- KPI tracking
- Exploratory Data Analysis (EDA)
- Dashboard development
- Business insight generation

## Business Problem

Management required visibility into:
- Top-performing product categories
- Regional sales performance
- Revenue contribution trends
- Customer purchasing behavior
- Operational KPIs

The goal was to transform raw sales data into meaningful insights that support strategic and operational decision-making.

## Dataset Information

The dataset contains consumer goods sales records including:
- Product categories
- Sales transactions
- Regional performance
- Revenue metrics
- Customer purchase data

The dataset was cleaned and transformed before analysis to ensure data consistency and accuracy.

## SQL Concepts Used

- Joins
- Common Table Expressions (CTEs)
- Aggregate Functions
- Window Functions
- Subqueries
- GROUP BY & HAVING
- Ranking Functions
- Filtering and Sorting

## Key Business Questions Solved

- Which product categories generated the highest revenue?
- Which regions contributed most to overall sales?
- What were the monthly and quarterly sales trends?
- Which products showed declining performance?
- Which customer segments contributed most to revenue?
- Which regions demonstrated consistent growth patterns?

## Project Workflow

1. Collected and reviewed raw sales datasets
2. Cleaned and transformed data using Excel and SQL
3. Performed Exploratory Data Analysis (EDA)
4. Wrote SQL queries to extract business insights
5. Built interactive Power BI dashboards
6. Analyzed KPIs and sales trends
7. Generated actionable insights and recommendations
